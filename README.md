# Arduino UNO Q Skill

## Install

**Global** (available in every project):

```bash
git clone https://github.com/maybethereisone-C/Arduino-UNO-Q-Skill.git ~/.claude/skills/arduino-uno-q
```

**Project-scoped** (one repo only):

```bash
git clone https://github.com/maybethereisone-C/Arduino-UNO-Q-Skill.git <your-project>/.claude/skills/arduino-uno-q
```

## Use

In Claude Code, invoke with the Skill tool:

```
arduino-uno-q
```

Start at `SKILL.md` — it routes to `references/` on demand.

## Update

```bash
cd ~/.claude/skills/arduino-uno-q   # or your project path
git pull
```
